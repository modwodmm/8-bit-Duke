#8-bit-Duke:
8-bit-Duke is a CLI game in which you help the java mascot duke to navigate through the deep world of tech.
##Chapters:
There are 4 total chapters in this game.
1. Memory leak
2. Cpu overheating
3. Updating PC
4. Cleaning storage
I wont describe the chapters in detail so I don't spoil the game.
##How to run:
1. Compile the program.
2. Run 'main.java'.
3. Enjoy the game.
